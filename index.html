Printing (Debug and Non-Debug Modes)

Debug mode helps in inspecting complex structures during development. It supports the following:

  • {:?} for single-line debug output.
  • {:#?} for multi-line, pretty debug output.
fn main() {
  println!("Value: {}", 42); // Non-Debug
  println!("Debug Value: {:?}", (42, "Rust")); // Debug
}

Intermediate

Enums with Error Handling

enum FileResult {
  Ok(String),
  Err(String),
}

fn open_file() -> FileResult {
  if true {
    FileResult::Ok("File opened successfully".to_string())
  } else {
    FileResult::Err("Failed to open file".to_string())
  }
}
